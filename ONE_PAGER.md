# Hotel Cash Flow Forecasting MCP Server

## AI-Powered Revenue Management for Hotels

Intelligent cash flow forecasting and dynamic pricing, integrated with Oracle Opera PMS and EPM Planning.

---

## Key Capabilities

| Feature | Description |
|---------|-------------|
| **Cash Flow Forecasting** | Real-time projections up to 90 days |
| **Dynamic Pricing** | AI-optimized rates based on 6 factors |
| **Opera Integration** | Sync rates directly to Oracle Opera PMS |
| **Event Intelligence** | Local events calendar with demand impact |
| **Competitor Analysis** | Market positioning vs Marriott, Hilton, Hyatt |
| **Scenario Planning** | What-if analysis for occupancy and rate changes |

---

## Dynamic Pricing Engine

| Factor | Impact |
|--------|--------|
| Occupancy | -25% to +50% |
| Day of Week | -12% (Tue) to +25% (Sat) |
| Seasonality | -20% (Jan) to +35% (Dec) |
| Lead Time | -15% (advance) to +30% (same day) |
| Local Events | +15% to +60% |
| Competitor Rates | Capped at 15% above market |
